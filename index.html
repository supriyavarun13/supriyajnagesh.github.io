Project 01

Sai Dance Academy — Customer Portal

Experience Cloud
What I built

A full-stack customer portal for Sai Dance Academy using Salesforce Experience Cloud. Students can browse classes, enrol online, and pay via Stripe — all from a public-facing LWR website. Built with custom LWC components, Apex REST integration, record-triggered Flows, and a custom data model spanning five objects. Demonstrates end-to-end Salesforce development from requirements gathering through deployment.

Project 02

SLA Command Centre

Service Cloud
Problem statement

Sai Dance Academy's support team had no early warning system — SLA breaches only surfaced in retrospective reports, after the damage was already done.

What I built

A fully declarative SLA monitoring system with real-time case health indicators, automated 75% warning alerts, escalation rules that notify managers instantly, and a live Service Cloud dashboard. SLA breach rate dropped by 60%.
